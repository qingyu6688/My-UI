## 2. 项目定位

My UI 是一个参考 Element Plus 工程组织方式的 Vue 3 组件库。项目面向中后台系统、管理端页面、工具型产品和业务组件沉淀场景，目标是提供一套视觉统一、类型清晰、主题可定制的基础组件。

项目创新点是运行时主题切换。用户可以修改主色、辅助色和强调色，并立即应用到按钮、表单、反馈、导航和数据展示组件中。

## 3. 默认主题

| 颜色 | 色值 | 用途 |
| --- | --- | --- |
| 鼠尾草主色 | `#a8d5ba` | 主按钮、选中态、焦点态、链接强调 |
| 浅绿辅助色 | `#d7e8bd` | 辅助背景、成功状态、轻量提示 |
| 浅黄强调色 | `#fff3c7` | 温和提示、弱警告、空状态背景 |

## 4. 目标用户

- 前端开发者：在 Vue 3 项目中快速搭建统一 UI。
- 中后台项目团队：需要稳定组件、统一主题和较低维护成本。
- 设计和产品协作者：需要通过主题色快速验证品牌风格。
- 组件库维护者：需要清晰的工程结构、文档、测试和发布流程。

## 6. 当前组件范围

### 基础组件

| 组件 | 说明 |
| --- | --- |
| Button 按钮 | 触发页面操作，支持类型、尺寸、朴素、圆角、文本、图标和加载状态。 |
| Color 色彩 | 展示默认主题色、语义色和运行时主题变量。 |
| Container 容器 | 页面级布局容器，适合 Header、Aside、Main、Footer 组合。 |
| Divider 分割线 | 分隔内容块，支持水平、垂直和带文字的分隔场景。 |
| Icon 图标 | 统一图标尺寸、颜色和可访问性标识，内置 80 个常用图标并提供搜索、网格展示和复制组件名能力。 |
| Layout 布局 | 栅格或结构化布局能力，适合页面分区和响应式排版。 |
| Link 链接 | 用于页面跳转或外部链接，支持不同语义类型。 |
| Space 间距 | 处理按钮组、标签组和行内元素之间的稳定间距。 |
| Text 文本 | 统一正文、辅助说明、状态文案和截断展示。 |

### 配置组件

| 组件 | 说明 |
| --- | --- |
| ConfigProvider 全局配置 | 统一管理命名空间、主题、尺寸、层级和组件默认配置。 |

### 表单组件

| 组件 | 说明 |

## 7. 主题系统需求

| 能力 | 优先级 | 说明 |
| --- | --- | --- |
| 默认主题 | P0 | 内置鼠尾草浅黄主题 |
| 运行时切换 | P0 | 修改 CSS Variables 后页面立即生效 |
| 主题生成 | P0 | 根据三类基础色生成 hover、active、light 等派生色 |
| 主题重置 | P0 | 一键恢复默认主题 |
| 主题持久化 | P1 | 保存最近一次主题配置 |
| 主题导出 | P1 | 导出 JSON 或 CSS Variables |
| 主题导入 | P1 | 导入 JSON 后恢复主题 |
| 对比度提示 | P2 | 颜色对比不足时给出提示 |

## 8. 文档站需求

- 首页：展示项目定位、默认主题、组件入口和路线概览。
- 指南：说明安装、快速开始、按需引入和主题定制。
- 组件：按分类展示组件说明、使用示例和 API 链接。
- 演练场：提供独立在线调试页面，支持示例代码编辑、运行、重置、复制、自动运行和实时预览。
- 主题：提供可视化主题编辑器和实时预览。
- 路线：说明当前完成项、下一步和计划中能力。
- 项目文档：保留 API、数据库 Schema、部署说明、环境记录、更新日志。

## 10. 验收标准

- `npm run typecheck` 通过。
- `npm run lint` 通过。
- `npm test` 通过。
- `npm run build` 通过。
- `npm run docs:build` 通过。
- 默认主题色准确使用 `#a8d5ba`、`#d7e8bd`、`#fff3c7`。
- 文档站导航、组件页和主题编辑器在桌面和移动端都能正常浏览。
