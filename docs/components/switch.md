# Switch 开关

表达启用/停用等即时状态切换。

## 基础用法

```vue
<template>
  <my-switch>Switch</my-switch>
</template>
```

## 引入方式

```ts
import { MySwitch } from '@qingyu6688/my-ui'
```

## 设计说明

- 分类：表单组件。
- 样式会跟随 My UI 主题变量变化。
- 复杂交互需要同时关注默认、悬停、聚焦、禁用和错误状态。
- 后续会继续补齐 Props、Events、Slots 和 Exposes 表格。
