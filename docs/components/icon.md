# Icon 图标

统一图标尺寸、颜色和可访问性标识。

## 基础用法

```vue
<template>
  <my-icon>Icon</my-icon>
</template>
```

## 引入方式

```ts
import { MyIcon } from '@qingyu6688/my-ui'
```

## 设计说明

- 分类：基础组件。
- 样式会跟随 My UI 主题变量变化。
- 复杂交互需要同时关注默认、悬停、聚焦、禁用和错误状态。
- 后续会继续补齐 Props、Events、Slots 和 Exposes 表格。
