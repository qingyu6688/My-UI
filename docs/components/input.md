# Input 输入框

文本输入基础控件，支持前后缀、清空、禁用和状态展示。

## 基础用法

```vue
<template>
  <my-input>Input</my-input>
</template>
```

## 引入方式

```ts
import { MyInput } from '@qingyu6688/my-ui'
```

## 设计说明

- 分类：表单组件。
- 样式会跟随 My UI 主题变量变化。
- 复杂交互需要同时关注默认、悬停、聚焦、禁用和错误状态。
- 后续会继续补齐 Props、Events、Slots 和 Exposes 表格。
