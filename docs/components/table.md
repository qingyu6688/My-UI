# Table 表格

结构化展示数据，支持列配置和边框展示。

## 基础用法

```vue
<template>
  <my-table>Table</my-table>
</template>
```

## 引入方式

```ts
import { MyTable } from '@qingyu6688/my-ui'
```

## 设计说明

- 分类：数据展示。
- 样式会跟随 My UI 主题变量变化。
- 复杂交互需要同时关注默认、悬停、聚焦、禁用和错误状态。
- 后续会继续补齐 Props、Events、Slots 和 Exposes 表格。
