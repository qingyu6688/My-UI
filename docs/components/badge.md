# Badge 徽章

用于数字提醒、状态点和轻量角标。

## 基础用法

```vue
<template>
  <my-badge>Badge</my-badge>
</template>
```

## 引入方式

```ts
import { MyBadge } from '@qingyu6688/my-ui'
```

## 设计说明

- 分类：数据展示。
- 样式会跟随 My UI 主题变量变化。
- 复杂交互需要同时关注默认、悬停、聚焦、禁用和错误状态。
- 后续会继续补齐 Props、Events、Slots 和 Exposes 表格。
