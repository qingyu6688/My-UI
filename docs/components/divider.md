# Divider 分割线

分隔内容块，支持水平、垂直和带文字的分隔场景。

## 基础用法

```vue
<template>
  <my-divider>Divider</my-divider>
</template>
```

## 引入方式

```ts
import { MyDivider } from '@qingyu6688/my-ui'
```

## 设计说明

- 分类：基础组件。
- 样式会跟随 My UI 主题变量变化。
- 复杂交互需要同时关注默认、悬停、聚焦、禁用和错误状态。
- 后续会继续补齐 Props、Events、Slots 和 Exposes 表格。
