# Message 消息提示

展示短时反馈，适合保存、删除和校验结果。

## 基础用法

```vue
<template>
  <my-message>Message</my-message>
</template>
```

## 引入方式

```ts
import { MyMessage } from '@qingyu6688/my-ui'
```

## 设计说明

- 分类：反馈组件。
- 样式会跟随 My UI 主题变量变化。
- 复杂交互需要同时关注默认、悬停、聚焦、禁用和错误状态。
- 后续会继续补齐 Props、Events、Slots 和 Exposes 表格。
