# Button 按钮

用于触发操作。

## 基础用法

<my-button>默认按钮</my-button>
<my-button type="primary">主要按钮</my-button>
<my-button type="primary" plain>朴素按钮</my-button>

```vue
<template>
  <my-button>默认按钮</my-button>
  <my-button type="primary">主要按钮</my-button>
  <my-button type="primary" plain>朴素按钮</my-button>
</template>
```
