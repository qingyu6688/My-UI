# Tabs 标签页

在同级内容之间切换，适合示例、配置和详情分组。

## 基础用法

```vue
<template>
  <my-tabs>Tabs</my-tabs>
</template>
```

## 引入方式

```ts
import { MyTabs } from '@qingyu6688/my-ui'
```

## 设计说明

- 分类：导航。
- 样式会跟随 My UI 主题变量变化。
- 复杂交互需要同时关注默认、悬停、聚焦、禁用和错误状态。
- 后续会继续补齐 Props、Events、Slots 和 Exposes 表格。
