# Select 选择器

从固定选项中选择内容，适合表单筛选和配置项。

## 基础用法

```vue
<template>
  <my-select>Select</my-select>
</template>
```

## 引入方式

```ts
import { MySelect } from '@qingyu6688/my-ui'
```

## 设计说明

- 分类：表单组件。
- 样式会跟随 My UI 主题变量变化。
- 复杂交互需要同时关注默认、悬停、聚焦、禁用和错误状态。
- 后续会继续补齐 Props、Events、Slots 和 Exposes 表格。
