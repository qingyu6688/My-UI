# Tooltip 文字提示

为图标按钮、短文本和不可见信息补充说明。

## 基础用法

```vue
<template>
  <my-tooltip>Tooltip</my-tooltip>
</template>
```

## 引入方式

```ts
import { MyTooltip } from '@qingyu6688/my-ui'
```

## 设计说明

- 分类：反馈组件。
- 样式会跟随 My UI 主题变量变化。
- 复杂交互需要同时关注默认、悬停、聚焦、禁用和错误状态。
- 后续会继续补齐 Props、Events、Slots 和 Exposes 表格。
