# Avatar 头像

展示用户、组织或对象的头像、文字和图标占位。

## 基础用法

```vue
<template>
  <my-avatar>Avatar</my-avatar>
</template>
```

## 引入方式

```ts
import { MyAvatar } from '@qingyu6688/my-ui'
```

## 设计说明

- 分类：数据展示。
- 样式会跟随 My UI 主题变量变化。
- 复杂交互需要同时关注默认、悬停、聚焦、禁用和错误状态。
- 后续会继续补齐 Props、Events、Slots 和 Exposes 表格。
