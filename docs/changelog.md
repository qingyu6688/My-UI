## 1.0.9 - 2026-06-10

- 新增 `useSize` 组合式函数，按 **组件 prop > Form 父级 > ConfigProvider 全局 > default** 优先级解析尺寸。
- Button / Tag / Switch / Segmented / InputNumber 接入全局 size，被 `<my-config-provider size="large">` 包裹时统一放大。
- Form 自身、Input / Select、Radio-Group / Checkbox-Group 也接入：Form 内部子组件自动跟随 Form 的 size，Form 未指定时回退到 ConfigProvider 全局。
- 所有上述组件 size 默认值改为 `undefined`（之前是 `'default'`），不再"截断"全局配置；prop 类型保持不变，向下兼容。
- 版本号从 1.0.8 升级到 1.0.9。
