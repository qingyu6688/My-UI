# 更新日志

## 1.0.3 - 2026-06-07

- 新增一批增强组件目录和样式入口，组件模块导出扩展到 75 个。
- 更新组件画廊页面和首页组件数量展示，覆盖基础、表单、数据展示、导航、反馈和其他组件分类。
- 修正 `layout.scss` 中 Sass 百分比函数写法，减少构建弃用提示。
- 版本号从 1.0.2 升级到 1.0.3。

## 1.0.2 - 2026-06-06

- 重写 README 和 docs 下的核心 Markdown 文档，修复中文乱码问题。
- 按当前项目结构重建组件文档页面，覆盖 45 个组件目录和色彩说明。
- 重写 VitePress 导航和侧边栏文案，文档站结构更贴近组件库使用路径。
- 补充 API、部署、数据库 Schema、环境检查和需求文档。
- 版本号从 1.0.1 升级到 1.0.2。

## 1.0.1 - 2026-06-06

- 新增首页、指南、组件、主题、路线五个独立路由页面。
- 优化顶部导航栏，支持路由高亮、搜索跳转和移动端折叠菜单。
- 调整首页为 Element Plus 风格的文档站入口页。
- 新增组件分类页、主题编辑页和路线规划页的基础版式。
- 新增 `vue-router` 作为文档站页面路由依赖。

## 1.0.0 - 2026-06-06

- 初始化 Vue 3 + TypeScript + Vite 组件库工程。
- 新增默认鼠尾草浅黄主题变量。
- 新增运行时主题 API：`createTheme`、`applyTheme`、`useTheme`。
- 新增 `MyButton` 示例组件。
- 新增 VitePress 文档站基础结构。
